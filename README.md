# Приложение библиотека фильмов
Тема: Добавление фильмов (название, год, рейтинг, жанр), фильтрация по жанру или году.
## User stories:
Как пользователь, я хочу иметь возможность добавлять новые фильмы и просматривать старые.
Как пользователь, я хочу иметь возможность фильтровать фильмы по жанру.
Как пользователь, я хочу иметь возможность сортировать фильмы по году выпуска.
Как пользователь, я хочу иметь возможность изменять и удалять существующие фильмы.
## Минимальный работающий продукт (MVP):
В рамках MVP реализуется консольное приложение, которое:
* Сохраняет фильмы при помощи sqlite.
* Позволяет добавлять новые фильмы, изменять и удалять фильмы.
* Фильтровать фильмы по одному критерию.
* Сортировать фильмы по одному критерию.
