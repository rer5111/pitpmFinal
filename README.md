# Приложение библиотека фильмов
Тема: Добавление фильмов (название, год, рейтинг, жанр), фильтрация по жанру или году.
## User stories:
* Как пользователь, я хочу иметь возможность добавлять новые фильмы и просматривать старые.
* Как пользователь, я хочу иметь возможность фильтровать фильмы по жанру.
* Как пользователь, я хочу иметь возможность сортировать фильмы по году выпуска.
* Как пользователь, я хочу иметь возможность изменять и удалять существующие фильмы.
* Как пользователь, я хочу иметь возможность вводить неверные данные без нарушения работы программы.
## Минимальный работающий продукт (MVP):
В рамках MVP реализуется консольное приложение, которое:
* Сохраняет фильмы при помощи sqlite.
* Позволяет добавлять новые фильмы, изменять и удалять фильмы.
* Фильтровать фильмы по одному критерию.
* Сортировать фильмы по одному критерию.
## Структура данных:
Данные хранятся в sqlite файле db.db.
Фильм (film_model.py):
* id: int
* title: string
* year: int
* rating: float
* genre: string
## Архитектура
Функционал в проекте разделен на 2 уровня.
* /src - main.py и menu.py - Входная точка и меню приложения
* /src/db - film_model.py и movie_dv.py - Модель фильма и файл с CRUD действиями БД.
Также в папке /tests хранятся:
* test_crud - тесты функций в /src/db
* test_menu - тесты функций в /src
* test_edge_cases - тесты крайних случаев во всем проекте
```
Пользователь (main.py)
        |
        V
   Меню (menu.py)
        |
        V
CRUD-функции (movie_db.py)
        |
        V
База данных (db.db)
```
## Запуск приложения
Для запуска приложения необходимо открыть терминал в корневой папке приложение и прописать команду:
```
python -m src.main
```
Для запуска тестов необходимо открыть терминал в корневой папке приложения и прописать команду:
```
pytest tests
```
