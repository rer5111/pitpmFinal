# Чек-лист приемочного тестирования
1. Добавление фильма
- Шаги: Нажать 2, ввести название, год выпуска, жанр и рейтинг фильма.
- Результат: Сообщение с ID фильма.
- Статус: Пройден 
2. Изменение фильма
- Шаги: Нажать 3, ввести ID существующего фильма.
- Результат: Сообщение "Фильм успешно обновлен!"
- Статус: Пройден
3. Удаление фильма
- Шаги: Нажать 4, ввести ID существующего фильма.
- Результат: Сообщение "Фильм успешно удален!"
- Статус: Продйен
4. Вывод всех фильмов
- Шаги: Нажать 1.
- Результат: Сообщение со всеми фильмами.
- Статус: Пройден
5. Вывод фильмов с фильтром по жанру
- Шаги: Нажать 5, ввести жанр, присутствующий у существующего фильма
- Результат: Сообщение фильмами этого жанра.
- Статус: Пройден
6. Вывод фильмов с сортировкой года по возрастанию
- Шаги: Нажать 6.
- Результат: Сообщение с фильмами от старейшего до новейшего.
- Статус: Пройден
7. Вывод фильмов с сортировкой года по убыванию
- Шаги: Нажать 7.
- Результат: Сообщение с фильмами от новейшего до старейшего.
- Статус: Пройден
8. Работа программы при вводе неправильного пункта в меню
- Шаги: Нажать 9.
- Результат: Сообщение "Данная функция отсутствует в меню"
- Статус: Пройден
9. Работа программы при вводе текста в числовом поле
- Шаги: Нажать 2, ввести название, год выпуска, жанр и рейтинг фильма.
- Результат: Сообщение с ID фильма.
- Статус: Пройден
10. Вывод пустого списка при отсутствии фильмов
- Шаги: Удалить все фильмы, нажать 1.
- Результат: Сообщение "Фильмов нет"
- Статус: Пройден
